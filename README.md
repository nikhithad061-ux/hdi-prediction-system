# 🌍 HDI Prediction System – A Comprehensive Measure of Well-being

## 📌 Project Overview
This project predicts the **Human Development Index (HDI)** using Machine Learning. It evaluates a country's development level based on key indicators such as life expectancy, education, and income, and provides both numerical prediction and classification.

---

## 🎯 Problem Statement
Analyzing human development is complex because it depends on multiple factors like health, education, and income. Manual analysis is time-consuming and inefficient.  
This project provides an automated system to predict HDI values and classify development levels.

---

## 💡 Solution
A Machine Learning-based web application that:
- Accepts user input (life expectancy, education, income)
- Predicts HDI using Linear Regression
- Classifies output into Low, Medium, High, and Very High categories

---

## ✨ Features
- HDI Prediction using ML model  
- User-friendly Web Interface  
- Data Visualization (Graphs & Heatmaps)  
- Development Classification  
- Real-time Predictions
